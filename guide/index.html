# Getting Started

# Installing

  1. Simply clone a wm-go repository somewhere in your projects directory
$ git clone https://github.com/artemvasilkin/wm-go.git
$ cd wm-go
$ npm link

Now all commands from wm-go is available to use in any directory.

WARNING

Please, make sure you have blocks repository as well as wm-cli package pre-installed on your computer.

  1. Add users information and github token
$ wm-config

wm-config calls a cli interface for entering login info.

# Usage

After your wm-config is set up you can open your blocks repository and start working. For example

$ cd blocks
$ wm-open 1/services

# Updating wm-go

Just pull changes from git and relink commands

$ cd wm-go
$ git pull
$ npm unlink
$ npm link

# Removing wm-go

You'll need to unlink commands and remove repository from your computer

$ cd wm-go
$ npm unlink
$ cd ../
$ rm -rf wm-go
